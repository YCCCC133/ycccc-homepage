# 劳动争议纠纷民事起诉状模板（2025版）

## 一、填写说明

为了方便您更好地参加诉讼，保护您的合法权利，请填写本表。

1. 起诉时需向人民法院提交证明您身份的材料，如身份证复印件、营业执照复印件等。
2. 本表所列内容是您提起诉讼以及人民法院查明案件事实所需，请务必如实填写。
3. 本表有些内容可能与案件无关。与案件无关的项目可以填写“无”或不填；勾选项可在对应项打“√”；如另有重要内容，可另附页填写。
4. 电子版填写时，相关栏目可复制粘贴或扩容，但不得改变要素内容和格式设置。例如，多原告、多被告或多委托诉讼代理人等情况，可按实际需要复制扩展。

## 二、特别提示

- 诉讼参加人应遵守诚信原则，如实认真填写表格。
- 如存在虚假诉讼、恶意诉讼、滥用诉权等情形，人民法院将依法追究责任。
